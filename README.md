# AI Engineering Fundamentals

A collection of projects and resources for learning AI engineering fundamentals. This includes coursework from [CodePath](https://www.codepath.org/) as well as general AI engineering resources, guides, and tools picked up along the way.

## Projects

### [Game Glitch Investigator](https://github.com/Akhan521/Game-Glitch-Investigator)

A debugging exercise where an AI-generated Streamlit number guessing game ships completely broken: wrong hints, resetting state, and flawed logic. The goal is to investigate the bugs, fix them, refactor the code for testability, and get all tests passing.

### [PawPal+](https://github.com/Akhan521/PawPal)

A Streamlit app for planning daily pet care tasks across multiple pets. Features a greedy scheduling algorithm, recurring task support, conflict detection, and a full interactive UI, designed UML-first and implemented with a test-driven approach.
